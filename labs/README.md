# AWS Certified Data Engineer Associate Lab Guide

This folder contains practical labs for TGS-2025053209 - AWS Certified Data Engineer Associate Training.

## Lab Sequence

| Lab | File | Outcome |
| --- | --- | --- |
| 1 | [Account Safety, IAM, S3 Data Lake, and Data Formats](lab-01-account-iam-s3-data-lake.md) | Secure account baseline and S3 data lake structure with encryption and data-format notes. |
| 2 | [Data Ingestion with Kinesis, DMS, Transfer, and Event Patterns](lab-02-data-ingestion-kinesis-dms-transfer.md) | Ingestion pattern comparison for batch, streaming, migration, and file-transfer scenarios. |
| 3 | [Glue Data Catalog, Crawlers, Athena, and Schema Discovery](lab-03-glue-catalog-crawlers-athena.md) | Glue catalog database, crawler workflow, Athena queries, and partition awareness. |
| 4 | [Glue ETL, Data Quality, and Data Transformation](lab-04-glue-etl-data-quality.md) | ETL job design, transformation outputs, and data quality checklist. |
| 5 | [Data Modeling with Redshift, DynamoDB, OpenSearch, and RDS](lab-05-redshift-dynamodb-opensearch-data-modeling.md) | Service selection and data model comparison for common analytical and operational workloads. |
| 6 | [Pipeline Orchestration with Step Functions, MWAA, and EventBridge](lab-06-step-functions-mwaa-eventbridge-orchestration.md) | Pipeline orchestration design with dependencies, schedules, retries, and failures. |
| 7 | [Security, Governance, Lake Formation, KMS, and Data Lifecycle](lab-07-security-governance-lake-formation-kms.md) | Data permissions, encryption, lifecycle, retention, and governance practice. |
| 8 | [Monitoring, Cost Optimization, Reliability, and Exam Review](lab-08-monitoring-cost-optimization-exam-review.md) | Monitoring plan, cost controls, reliability checklist, and timed exam-style review. |
